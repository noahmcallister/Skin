Buttons

Standalone Buttons

Buttons can be created as a hyperlink or div with class button, or as an actual <button> element. There are a few default button classes to help with commonly used styles, but buttons can be completely customized using bfm-customize.css classes.

Default buttons

Hyperlink Button
Div Button

							<a class="button" href="#" title="Button">Hyperlink Button</a>
							<div class="button">Div Button</div>
							<button>Button</button>
