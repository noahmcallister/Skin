Customization

The following classes are all found in the bfm-customize.css file, and can be used to customize elements in a wide variety of ways.

Float

Class Description
.float-left Left float element
.float-left-wrap Left float element with margins to wrap text around it
.float-right Right float element
.float-right-wrap Right float element with margins to wrap text around it

Width

Widths can be customized by using class width-{breakpoint}-{percent}. Breakpoints correspond to those used by the grid.

Class Description
.width-25 25% width
.width-33 33% width
.width-50 50% width
.width-66 66% width
.width-75 75% width
.width-100 100% width

Max width

Class Description
.max-width-25 Max 25% width
.max-width-33 Max 33% width
.max-width-50 Max 50% width
.max-width-66 Max 66% width
.max-width-75 Max 75% width
.max-width-100 Max 100% width

Padding

The following classes can be added to an element to give it a customized padding. Adding one of the following classes to a .row will apply the padding to it's columns.

Class Description
.padding0 No padding
.padding0-25 0.25em padding
.padding0-50 0.5em padding
.padding0-75 0.75em padding
.padding1-0 1em padding

Margin

Class Description
.margin0 No margin
.margin0-25 0.25em margin
.margin0-50 0.5em margin
.margin0-75 0.75em margin
.margin1-0 1em margin

Easing

Class Description
.ease-opacity 0.3s opacity transition
.ease-color 0.3s color transition
.ease-border 0.3s border transition
.ease-bg 0.3s background transition
.ease-all 0.3s all transition

Class Description

Hover Zoom

To create a container which hides the zooming overflow, wrap the element which contains one of the following classes with class hover-zoom-container.

Class Description
.hover-zoom25 0.3s zoom to 25%
.hover-zoom50 0.3s zoom to 50%

Color Customization

Types

All of the example colors shown below can be used with backgrounds, text, and borders.

Class Description
.bg-{color} Background color
.hover-bg-{color} Background color (on hover)
.color-{color} Font color
.hover-color-{color} Font color (on hover)
.border-{color} Border color
.hover-border-{color} Border color (on hover)
