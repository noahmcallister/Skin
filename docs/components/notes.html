The following examples requires use of the jQuery library:

Dismiss

Any element with class msg-dismiss will dismiss the element whose id matches it's data-dismiss attribute.

This is an error message 

							<div id="error-message" class="message box-error"><i class="fal fa-times-circle"></i> This is an error message <span class="msg-dismiss" data-dismiss="error-message"><i class="fal fa-times-circle"></i></span></div>

A dismissal can be "remembered" by setting a cookie on the user's computer so that the message will not reappear for a set number of hours. This is done by adding a data-remember attribute whose value represents the number of hours to dismiss the message. A value of 0 will dismiss the message for a period of one year.

This is a success message

							<div id="success-message" class="message box-success"><i class="fal fa-check-circle"></i> This is a success message<span class="msg-dismiss" data-dismiss="success-message" data-remember="24"><i class="fal fa-times-circle"></i></span></div>
