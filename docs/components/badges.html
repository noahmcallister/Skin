Badges

Badges are typically used to alert a user to some sort of notification, but can also be used in a wide variety of other ways.

Example

This is a badge within a paragraph. New 9

This is a badge within a button: Notifications 4 Notifications 4

  • New Some new feature was added
  • Bug Bug was fixed
