Javascript Classes

There are a variety of classes that can be added to elements which are controlled by Javascript.

Class Trigger Description
.js-hide ready() Javascript will hide the element
.js-show ready() Javascript will show the element
.set-cookie click() Sets a cookie on the client's browser. Needs data attributes data-cookie-name, data-cookie-value, data-cookie-duration (in hours). Duration of 0 will set the duration for 1 year.
.click-delay click() Delays the firing of the click() by the set number of seconds. Needs data attributes data-delay (in seconds)
.click-auto ready() Automatically triggers a click() when the DOM is loaded.
.smooth-scroll click() Animated scroll to the position of href on the page.
.above-fold ready() Automatically sets the max-height of the element to the height of the browser window, up to 800px
#top click() Any element with a href="#top" will animate the scroll to the top of the page. *

* To add an element that appears as the window is scrolled down, add an element with id top-button.


							<a id="top-button" href="#top" title="Scroll to top"><span class="fa-stack"><i class="fal fa-circle fa-stack-2x"></i><i class="fal fa-arrow-to-top fa-stack-1x"></i></span></a>
