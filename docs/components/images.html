Hover Zoom

Images are a great way to utilize the hover-zoom classes. These classes include:

Class Description
.hover-zoom-container Container
.hover-zoom10 Zoom 10% on hover
.hover-zoom25 Zoom 25% on hover
.hover-zoom50 Zoom 50% on hover

This box utilizes several classes that are found in the bfm-customize.css file, which enable you to create all sorts of custom layouts.


							<div class="row no-gutters">
	
								<div class="col-md-6 offset-md-3 border-solid border-all1 border-main">
		
									<div class="hover-zoom-container">
								
										<img class="width100 hover-zoom25" src="img/web-design.jpg" alt="Image" />
								
									</div> <!-- /.hover-zoom-container -->
							
									<div class="padding1-0">
								
										<p class="em0-8 no-bottom">This box utilizes several classes that are found in the bfm-customize.css file, which enable you to create all sorts of custom layouts.</p>
								
									</div>
	
								</div> <!-- /.col-md-6 -->
		
							</div> <!-- /.row -->
