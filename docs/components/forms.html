Input element
Forms

Inputs

Input with no label


									<input type="text" id="text1" name="text_input1" placeholder="Text input" />

Input with label


									<input type="text" id="text2" name="text_input2" placeholder="Text input" />

Disabled input


									<input type="text" id="text3" name="text_input3" value="Text input" disabled>

Input error


									<input type="text" class="input-error" id="text4" name="text_input4" placeholder="Text input">

File input


									<label for="file1">File upload</label>
									<input type="file" id="file1" name="file_1" />

Radio

Labels for radio buttons must go after the input.


									<input type="radio" name="radio1" id="radio1_1" value="First" />
									<label for="radio1_1">First</label>
			
									<input type="radio" name="radio1" id="radio1_2" value="Second" />		
									<label for="radio1_2">Second</label>
			
									<input type="radio" name="radio1" id="radio1_3" value="Third" />
									<label for="radio1_3">Third</label>
			
									<input type="radio" name="radio1" id="radio1_4" value="Fourth" />
									<label for="radio1_4">Fourth</label>

To align horizontally on the same row for medium screen widths and above, wrap in class form-same-line.


									<div class="form-same-line">
				
										<input type="radio" name="radio2" id="radio2_1" value="First" />
										<label for="radio2_1">First</label>
					
										<input type="radio" name="radio2" id="radio2_2" value="Second" />
										<label for="radio2_2">Second</label>
				
										<input type="radio" name="radio2" id="radio2_3" value="Third" />
										<label for="radio2_3">Third</label>
			
										<input type="radio" name="radio2" id="radio2_4" value="Fourth" />
										<label for="radio2_4">Fourth</label>
				
									</div> <!-- /.form-same-line -->

Checkbox

Labels for checkboxes must go after the input.


									<input type="checkbox" name="checkbox1" id="checkbox1_1" value="First" />
									<label for="checkbox1_1"> First</label>
			
									<input type="checkbox" name="checkbox1" id="checkbox1_2" value="Second" />
									<label for="checkbox1_2"> Second</label>
			
									<input type="checkbox" name="checkbox1" id="checkbox1_3" value="Third" />
									<label for="checkbox1_3"> Third</label>
			
									<input type="checkbox" name="checkbox1" id="checkbox1_4" value="Fourth" />
									<label for="checkbox1_4"> Fourth</label>

To align horizontally on the same row for medium screen widths and above, wrap in class form-same-line.


									<div class="form-same-line">
									
										<input type="checkbox" name="checkbox2" id="checkbox2_1" value="First" />
										<label for="checkbox2_1"> First</label>
			
										<input type="checkbox" name="checkbox2" id="checkbox2_2" value="Second" />
										<label for="checkbox2_2"> Second</label>
			
										<input type="checkbox" name="checkbox2" id="checkbox2_3" value="Third" />
										<label for="checkbox2_3"> Third</label>
			
										<input type="checkbox" name="checkbox2" id="checkbox2_4" value="Fourth" />
										<label for="checkbox2_4"> Fourth</label>
								
									</div> <!-- /.form-same-line -->

Textareas

Add class md or lg to a textarea to make the default height larger.


									<label for="textarea1">Default text area</label>
									<textarea id="textarea1" name="text_area1" placeholder="Text area"></textarea>
								

									<label for="textarea2">Medium text area</label>
									<textarea class="md" id="textarea2" name="text_area2" placeholder="Text area"></textarea>
								

									<label for="textarea3">Large text area</label>
									<textarea class="lg" id="textarea3" name="text_area3" placeholder="Text area"></textarea>

Select

Selects must be wrapped inside of a select-wrap element in order to show the down arrow.


									<label for="select1">Select 1</label>
			
									<div class="select-wrap">
			
										<select id="select1" name="select_one">
											<option value="First">First</option>
											<option value="Second">Second</option>
											<option value="Third">Third</option>
											<option value="Fourth">Fourth</option>
										</select>
	
									</div> <!-- /.select-wrap-->

Fieldset Legend 

									<fieldset>
				
										<legend>Fieldset Legend</legend>
				
										<label for="text5">Text input</label>
										<input type="text" id="text5" name="text_5" placeholder="Text input" />

										<input type="text" id="text6" name="text_6" placeholder="Text input" />
				
									</fieldset>

Icon prepend/append

Wrap the form element and icon inside a form-icon-prepend or form-icon-append element.


									<label for="text7">Prepend icon</label>
			
									<div class="form-icon-prepend">
				
										<input id="text7" name="text_7" type="text" placeholder="Prepend icon" />
										<i class="fal fa-search"></i>
				
									</div> <!-- /.form-icon-prepend -->
								

									<label for="text8">Append icon</label>
			
									<div class="form-icon-append">
				
										<input id="text8" name="text_8" type="text" placeholder="Append icon" />
										<i class="fal fa-search"></i>
				
									</div> <!-- /.form-icon-append -->

Button append

Wrap the input and button inside a form-button-append element.


									<label for="text9">Text input</label>
			
									<div class="form-button-append">
			
										<input id="text9" name="text_9" type="text" placeholder="Text input" />
										<button type="submit"> <i class="fal fa-save"></i> Search</button>
				
									</div> <!-- /.form-button-append -->

Same line elements

Form elements will appear on the same line on medium screen widths and above by wrapping them in class form-same-line. Check window resizing to be sure nothing overflows. This works best for a single input and a button. For multiple inputs, it may be best to use columns. This will also give more control over the width of each input.


									<div class="form-same-line">
		
										<input id="text10" name="text_10" type="text" placeholder="Text input" /> <button type="submit"> <i class="fal fa-save"></i> Search</button>
				
									</div> <!-- /.form-same-line -->

Submit buttons


									<button type="submit"><i class="fal fa-save"></i> Submit button</button>
