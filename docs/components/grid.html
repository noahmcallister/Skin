Grid

The grid layout is heavily based on Bootstrap's grid, so you will notice many similarities. The grid layout is based on the principle that content is contained within a row, and each row contains up to 12 columns. For example, 12 columns would take up an entire row, 6 columns are half a row, 4 columns are a third of a row, and so on.

Breakpoints

Breakpoints include:

Breakpoint Description Example
{none} Display on all windows col-12
md Display on all windows 600px wide and above col-md-12
lg Display on all windows 900px wide and above col-lg-12
xl Display on all windows 1200px wide and above col-xl-12

Columns, columns inside columns

Each row equals total 12 columns. Column widths are specified by using col-1 to col-12. A 1rem left and right padding is added to each column unless the class no-gutters is added to the row.

Column offsets

Offsets can be used in place of columns. Each row still equals a total of 12 columns. Offsets are specified by using offset-1 to offset-12.

Columns will all automatically have the same height by default.

Horizontal alignment classes include justify-content-{breakpoint}-start, justify-content-{breakpoint}-center, justify-content-{breakpoint}-end, justify-content-{breakpoint}-around, justify-content-{breakpoint}-between. These classes must be added to the row.
