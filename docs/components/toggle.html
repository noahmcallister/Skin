Toggle

In the below example, the toggle box makes use of an icon, so the icon stylesheets must be loaded in order for it display. The toggle box also requires use of the jQuery library.

The main components of a toggle box are:

Class Description
.toggle-title This element's data-toggle attribute must equal the element ID to toggle
.toggle-icon .toggle-icon i will rotate 90 degrees when .toggle-title is clicked. (optional)
.toggle-content jQuery hides this element by default. It's ID must equal the data-toggle attribuite of the .toggle-title element. If Javascript is disabled, this element will be shown.
.toggle-open Adding this class to the toggle-content element will open the toggle by defult. (optional)
.toggle-all Clicking this element will toggle all the boxes at the same time. This element's html will be toggled with it's data-label attribute each time it is clicked. (optional)

What is the answer?

The answer is this.


							<div class="toggle-all color-light em0-8 pointer padding1-0" data-label="- Collapse all">+ Expand all</div>
						
							<h4 class="toggle-title" data-toggle="toggle1">What is the answer? <span class="toggle-icon"><i class="fal fa-fw fa-chevron-right"></i></span></h4>
					
							<div id="toggle1" class="toggle-content bg-main padding1-0"><p class="no-bottom">The answer is this.</p></div>
