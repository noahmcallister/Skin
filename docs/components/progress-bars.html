Progress Bar

CSS Only

Progress bars can be made using only CSS. In this manner, the width must be specified as an inline style:

Default progress bar:

With label:

65% Complete!

							<p>Default progress bar:</p>
					
							<div class="progress-bar">
								<div class="progress-amt" style="width:65%"></div>
							</div>
					
							<p>With label:</p>
				
							<div class="progress-bar">
								<div class="progress-amt" style="width:65%">65% Complete!</div>
							</div>

Javascript

When utilizing the jQuery library, animation and automatic percentage calculations can be applied to the .progress-amt element.

Attribuite Description
data-valuemin Minimum progress value (numeric, optional)
data-valuemax Maximum progress value (numeric, optional)
data-valuenow Current progress (numeric). If data-valuemin or data-valuemax are not set, this value will be used as a percentage.

							</div>
