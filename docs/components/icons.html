Icons

Although not necessary to use, we have included the FontAwesome icon pack, and examples throughout this documentation make use of these icons. The FontAwesome icons can be used by including the base stylesheet, as well as the stylesheets of any styles you wish to use.

Description Stylesheet
FontAwesome base vendor/fa/css/fontawesome.min.css
Brands vendor/fa/css/fa-brands.min.css
Light vendor/fa/css/fa-light.min.css
Regular css/fonts/fa/css/fa-regular.min.css
Solid css/fonts/fa/css/fa-solid.min.css 

							<link rel="stylesheet" href="vendor/fa/css/fontawesome.min.css" type="text/css" media="all" />
							<link rel="stylesheet" href="vendor/fa/css/fa-light.min.css" type="text/css" media="all" />

For more information, see the FontAwesome docs.
