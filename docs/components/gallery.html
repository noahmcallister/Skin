Image Gallery

Bayfront Bootstrap includes the lightGallery plugin to create image and video galleries. This also requires the use of the jQuery library.


							<link rel="stylesheet" href="vendor/lightgallery/css/lightgallery.css" media="all" />
							<script src="vendor/lightgallery/js/lightgallery.js"></script>
						
							<script>
								$(document).ready(function() {	
									$(".lightgallery").lightGallery();			
								});			
							</script>

In addition to the core plugin, additional lightGallery plugins are included with Bayfront Bootstrap. These can be used by including the following files:


							<script src="vendor/lightgallery/js/lg-fullscreen.js"></script>	
							<script src="vendor/lightgallery/js/lg-hash.js"></script>
							<script src="vendor/lightgallery/js/lg-share.js"></script>
							<script src="vendor/lightgallery/js/lg-video.js"></script>
							<script src="vendor/lightgallery/js/lg-thumbnail.js"></script>

For more information, see the lightGallery docs, or view it on GitHub.
