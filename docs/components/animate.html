Animate

Animation classes can be added to elements by including the following stylesheet:

Self-hosted


							<link rel="stylesheet" href="vendor/animate/animation.css" media="all" />

Bayfront Media CDN


							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/animate/animation.css" media="all" />

Immediate animation

To animate an element, add class animated {animation} where animation is an available animation. You may also want to include the class infinite for an infininte loop. If the element is not in view when the page loads and the animation is not set to an infinite loop, the animation may not be seen.

For more information, see the animate.css docs.

Example
Sign Up Now 

							<a class="em1-2 button bg-green hover-opacity color-white hover-color-white border-green animated zoomIn" href="#" title="Sign Up Now"><i class="fal fa-check"></i> Sign Up Now</a>

On events

Requirements: jQuery

On hover

To animate on hover, add the classes animated anim-hover along with the attribute data-anim="{type}" where {type} is the animation desired.

On click

To animate on click, add the classes animated anim-click along with the attribute data-anim="{type}" where {type} is the animation desired.

In view

Requirements: WOW.js

To animate an element when in view, add classes wow {type} where {type} is the animation desired.

Advanced options can be added to the element by means of data attributes:

Attribute Description
data-wow-duration="2s" The animation duration (in seconds)
data-wow-delay="5s" The delay before the animation starts (in seconds)
data-wow-offset="50" Offset from the bottom of the browser window before animation starts (in px)
data-wow-iteration="10" Number of times the animation is repeated

For more information, see the WOW.js docs.

Example
Sign Up Now 

							<a class="em1-2 button bg-green hover-opacity color-white hover-color-white border-green wow zoomIn" href="#" title="Sign Up Now"><i class="fal fa-check"></i> Sign Up Now</a>
