Prism.js

Bayfront Bootstrap includes the Prism.js plugin which can be used to stylize semantic markup for a variety of languages. This requires the use of the jQuery library.


							<link rel="stylesheet" href="vendor/prism/prism.css" media="all" />
							<script src="vendor/prism/prism.js"></script>

Simply add the class language-{language} to the code tag to apply the proper markup styling. Line numbers can be shown by adding the class line-numbers to the pre tag.


							function sayHello($name) {
								return 'Hello, '.$name;
							}
		
							echo sayHello('John');
						

							<pre class="line-numbers"><code class="language-php">
								function sayHello($name) {
									return 'Hello, '.$name;
								}
		
								echo sayHello('John');
							</code></pre>

For more information, see the Prism.js docs.
