Build a page

You can build your own page layout from scratch, starting with the most basic markup:


							<!doctype html>
							<html lang="en">
							
								<head>
									
									<meta charset="utf-8">
									<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
									
									<link rel="stylesheet" href="[STYLESHEET URL]" type="text/css" media="all" />
									
									<title>[PAGE TITLE]</title>
							
								</head>
								
								<body>
									
									<div id="wrapper">
									
										<div id="content" class="container">
											
											<!-- [PAGE CONTENT] -->
											
										</div> <!-- /#content -->
								
									</div> <!-- /#wrapper -->
										
								</body>
							
							</html>

The [STYLESHEET URL] referenced above is the Skin stylesheet as described here.

Container classes

There are several container classes to choose from, which are applied to the #content element:

Class Description
container 100% max-width
container-md 600px max-width
container-lg 900px max-width
container-xl 1200px max-width
container-article 32em max-width (approx 65 character lines)

Body classes

There are several classes that can be added to the body element which will help in styling the page in different ways:

Class Description
rounded Adds a rounded border radius to all the squared elements
edge Allows the page wrapper to extend all the way to the edge of the window (example)
flat Forces the entire page background to be the same color, giving it a flat appearance (example)
boxed Makes each container it's own individual box (example)
