Vendors

Some components and features of Skin are made possible by a carefully chosen group of vendor libraries. They are listed below.

Animate.css v3.5.2

CSS animations are made possible by Animate.css.

Self-hosted

							<link rel="stylesheet" href="vendor/animate/animation.css" media="all" />
Bayfront Media CDN

							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/animate/animation.css" media="all" />
Visit

Bootstrap v4.1.1

We rely on Bootstrap's bootstrap-reboot.min.css to "normalize" our environment before we start customizing it with our own styles. Our grid layout is also based heavily on that of Bootstrap.

Self-hosted

							<link rel="stylesheet" href="vendor/bootstrap/bootstrap-reboot.min.css" media="all" />
Bayfront Media CDN

							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/bootstrap/bootstrap-reboot.min.css" media="all" />
Visit

FontAwesome v5.0.12

Skin ships with multiple variations of the optional FontAwesome icon font, which allows you to use thousands of different icons on your site.

The base stylesheet must be included with any icon style:

Self-hosted

							<link rel="stylesheet" href="vendor/fa/css/fontawesome.min.css" media="all" />
Bayfront Media CDN

							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/fa/css/fontawesome.min.css" media="all" />

Then, add whichever icon styles you want to use:

Self-hosted

							<link rel="stylesheet" href="vendor/fa/css/fa-brands.min.css" media="all" />
							<link rel="stylesheet" href="vendor/fa/css/fa-light.min.css" media="all" />
							<link rel="stylesheet" href="vendor/fa/css/fa-regular.min.css" media="all" />
							<link rel="stylesheet" href="vendor/fa/css/fa-solid.min.css" media="all" />
Bayfront Media CDN

							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/fa/css/fa-brands.min.css" media="all" />
							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/fa/css/fa-light.min.css" media="all" />
							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/fa/css/fa-regular.min.css" media="all" />
							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/fa/css/fa-solid.min.css" media="all" />
Visit

jQuery v3.3.1

Much of the JavaScript functionality makes use of the jQuery library.

jQuery and other JavaScript should be referenced just before the closing </body> tag.

Recommended method

							<script data-cfasync="false" src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
Visit

Lightgallery v1.6.11

Skin uses the Lightgallery plugin for it's image and video galleries.

Self-hosted

							<link rel="stylesheet" href="vendor/lightgallery/css/lightgallery.css" media="all" />
							<script src="vendor/lightgallery/js/lightgallery.js"></script>
Bayfront Media CDN

							<link rel="stylesheet" href="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/css/lightgallery.css" media="all" />
							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lightgallery.js"></script>

Additional optional Lightgallery plugins are also available:

Self-hosted

							<script src="vendor/lightgallery/js/lg-fullscreen.js"></script>
							<script src="vendor/lightgallery/js/lg-hash.js"></script>
							<script src="vendor/lightgallery/js/lg-share.js"></script>
							<script src="vendor/lightgallery/js/lg-thumbnail.js"></script>
							<script src="vendor/lightgallery/js/lg-video.js"></script>
Bayfront Media CDN

							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lg-fullscreen.js"></script>
							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lg-hash.js"></script>
							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lg-share.js"></script>
							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lg-thumbnail.js"></script>
							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/lightgallery/js/lg-video.js"></script>
Visit

Bayfront Media CDN

WOW.js v1.1.3

WOW.js is used in conjunction with Animate.css in order to apply animations as elements are in view, and to add additional animation customizations.

Self-hosted

							<script src="vendor/wow/wow.min.js"></script>
Bayfront Media CDN

							<script src="//s1.bayfrontcdn.com/skin/latest/vendor/wow/wow.min.js"></script>
